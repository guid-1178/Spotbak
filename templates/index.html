Spotbak Setup

Spotbak is a simple app that is designed to backup your spotify playlists and save each one in a CSV file

Spotbak can be setup as a cron job and run on a regular basis.

This app needs a 'Client ID' and 'Client Secret' which requires a Spotify developer account in order to aquire these credentials.

Spotify for Developers

Please ensure you register the app and aquire credentials before continuing

This Spotbak is a self-hosted solution. All data is stored locally.
